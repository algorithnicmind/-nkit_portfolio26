# Project Overview: Ankit's Portfolio (2025 Edition)

## 1. Introduction

This project is a high-performance, modern personal portfolio website built to showcase Ankit's skills, projects, and contact information. It features a premium, dark-mode aesthetic with fluid animations and a dynamic AI-powered chatbot.

## 2. Core Objectives

- **Portfolio Showcase**: Display projects with details (tech stack, live links).
- **Interactive Experience**: Use animations (Framer Motion, GSAP) and 3D elements.
- **AI Chatbot**: A 100% database-driven chatbot that answers visitor questions using MongoDB.
- **Admin Dashboard**: Secure backend for managing content and analytics.
- **Optimization**: Fast load times, SEO-ready, and mobile responsive.

## 3. Key Design Philosophy

- **Glassmorphism**: Translucent cards and blurring effects (`backdrop-filter`).
- **Neumorphism/Gradients**: Subtle depth and vibrant color accents (`#6366f1` Indigo, `#ec4899` Pink).
- **Minimalism**: Clean typography (Outfit font) and whitespace.

## 4. Feature Summary

- **Hero Section**: Dynamic typing effect and social links.
- **About Section**: Interactive skills visualization.
- **Projects**: Grid layout with filtering.
- **Chatbot**: Floating widget with custom Q&A management.
- **Contact**: Secure form with Cloudflare Turnstile protection.
- **Admin Panel**: Login protected area to view messages and manage chatbot data.
